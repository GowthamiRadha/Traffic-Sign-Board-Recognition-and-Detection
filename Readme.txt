Install Python from python installer
Install modules opencv,numpy,scipy,tensorflow,keras
Download dataset from GTSRB website and load them in code folder.
	Link https://www.kaggle.com/meowmeowmeowmeowmeow/gtsrb-german-traffic-sign
For training of model which is constructed in trafficsignnet.py run train.py
model_weights1.h5 and model1.json file will be created.
To execute the project run test.py for GUI.
